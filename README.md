# AI Control Plane for OCI

Generated by NanoClaw/App Factory for Lionel Cave.

This repository contains a deployable **source scaffold** for an OCI AI Control Plane landing zone. It is designed for GitHub staging and OCI review, not automatic production deployment.

## Architecture

- **OCI API Gateway**: governed ingress for control-plane APIs
- **OKE**: long-running control-plane services
- **OCI Functions**: lightweight agent actions
- **LangGraph**: graph/agent orchestration runtime
- **LiteLLM**: model gateway and routing layer
- **OCI Generative AI**: default first-party model provider behind LiteLLM
- **Oracle MCP Gateway**: controlled Oracle tool/data access boundary
- **Oracle AI Database 26ai**: durable workflow state, checkpoints, memory, vector memory, and audit records
- **OCI Cache with Valkey 8.1**: caching, session acceleration, locks, idempotency, rate state
- **OCI DevOps + OCIR + Resource Manager/Terraform**: build, artifact, and infra backbone

## Safety Boundary

No secrets, tenancy OCIDs, private keys, tokens, or endpoint-specific values are committed. All credential values must be injected through approved OCI Vault / Resource Manager / OneCLI-managed references.

GitHub publishing does **not** imply OCI deployment. OCI deployment requires a separate explicit approval.
